# Bridge of Time

**CPT208 Human-Centric Computing | Group B4-3 | AY2025-26 S2**

Bridging Generations Through Playful Shared Experiences

## About

Bridge of Time is a mobile web application designed to strengthen emotional bonds between grandparents and grandchildren through playful, culturally-rooted shared experiences — such as the Chinese New Year's Eve dinner.

**Topic**: B4 - Relation between Generations
